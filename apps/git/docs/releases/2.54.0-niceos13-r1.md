## Compatibility

This image is a NiceOS-based Bitnami-compatible migration target for historical
Bitnami-style Git container workflows.

Preserved contract:

- `/opt/bitnami`
- `/opt/bitnami/git/bin`
- `/opt/bitnami/common/bin`
- `/opt/bitnami/common/lib/libnss_wrapper.so`
- `/opt/bitnami/scripts/git/entrypoint.sh`
- `/bitnami/git`
- `BITNAMI_*` environment variables
- arbitrary UID support
- Git LFS system filter configuration
