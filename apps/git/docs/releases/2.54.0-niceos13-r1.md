# NiceOS Git 2.54.0-niceos13-r1

Initial NiceOS Application Container release for Git.

## Image

- docker.io/niceos/git:2.54.0-niceos13-r1
- docker.io/niceos/git:2.54.0

## Runtime

- NiceOS.Container 13
- Git 2.54.0
- Git LFS 3.7.1
- OpenSSH clients
- nss_wrapper arbitrary UID support
- Bitnami-compatible /opt/bitnami runtime layout

## Compatibility

This image is a NiceOS-based Bitnami-compatible migration target for historical
Bitnami-style Git container workflows.

Preserved contract:

- /opt/bitnami
- /opt/bitnami/git/bin
- /opt/bitnami/common/bin
- /opt/bitnami/common/lib/libnss_wrapper.so
- /opt/bitnami/scripts/git/entrypoint.sh
- /bitnami/git
- BITNAMI_* environment variables
- arbitrary UID support
- Git LFS system filter configuration

## Security posture

The final runtime image removes:

- package manager commands
- repository metadata
- sshd
- systemd
- compiler/build tools

## Smoke status

Validated:

- git --version
- git-lfs version
- git lfs env
- HTTPS git ls-remote
- arbitrary UID
- read-only rootfs with tmpfs mounts
